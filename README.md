Teste Prático para a AEC - Desenvolvedo C#
Este texto contém a solucão para o teste prático de desenvolvedor C#. O sistema vai consistir
em uma plataforma de gestão de endereços com autenticação de usuario.

Linguagens Utilizadas
.Backend:ASP.NET Core MVC
.Banco de Dados:SQL Sever(utilizando o ADO.NET como suporte)
.Frontend:HTML%, Rzor Views e Bootstrap 5
.Ferramenta:Visual Studio 2026

Funcionalidades Implementadas
.Autenticação:Sistema de login com a validação diretamente ligada com o banco de dados.
CRUD de Endereços:Listagem, Cadastros (novos) e a Exclusão de registros
Design:Interface responsiva e organizada com Bootstrap.

Configuração
Para o funcionamento do projeto localmente, altere a string de conexão nos Controllers para apontar para o seu servidor SQL Server local.
